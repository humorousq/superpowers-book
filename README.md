## 📚 关于本书

Superpowers 是一套为 AI 编码代理打造的完整软件开发工作流系统。本书将帮助你：

- 完全理解 Superpowers 的工作原理
- 掌握每个核心技能的使用方法
- 学会设计和实施自己的工作流
- 能够自定义和扩展 Superpowers

## ✨ 特色

- **实用导向** - 每个概念都配有实际案例
- **循序渐进** - 从基础到高级，逐步深入
- **最佳实践** - 融合行业最佳实践
- **跨平台支持** - 覆盖 Claude Code、Cursor、Codex 等主流平台

## 🚀 本地开发

```bash
# 克隆仓库
git clone https://github.com/humorousq/superpowers-book.git

# 进入目录
cd superpowers-book

# 安装依赖
npm install

# 启动开发服务器
npm run docs:dev

# 访问 http://localhost:5173
```

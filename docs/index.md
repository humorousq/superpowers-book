# Superpowers: AI 编码代理的完整工作流

欢迎使用 Superpowers 学习指南！

## 关于本书

这是一本帮助你掌握 Superpowers 系统的完整教程。Superpowers 是一套为 AI 编码代理打造的完整软件开发工作流系统，它基于一组可组合的"技能"（skills）构建。

### 本书特色

- **实用导向**：每个概念都配有实际案例
- **循序渐进**：从基础到高级，逐步深入
- **最佳实践**：融合行业最佳实践
- **跨平台支持**：覆盖 Claude Code、Cursor、Codex 等主流平台

## 目标读者

- 使用 AI 编码代理的开发者
- 希望提升 AI 辅助编程效率的团队
- 对 AI 代理工作流感兴趣的技术人员

## 如何阅读本书

### 新手入门

建议按顺序阅读：
1. [第一章：Superpowers 简介](/chapter-01) - 了解核心概念
2. [第二章：平台适配](/chapter-02) - 安装配置你的环境
3. [第三章：核心 Skills 详解](/chapter-03) - 掌握核心技能
4. [第四章：工作流深入理解](/chapter-04) - 实践完整流程

### 进阶学习

完成基础章节后：
- [第五章：设计文档解析](/chapter-05) - 理解设计思想
- [第六章：实施计划示例](/chapter-06) - 学习实际案例
- [第七章：测试与质量保证](/chapter-07) - 深入测试实践

### 高级应用

当你熟练掌握后：
- [第九章：高级主题与扩展](/chapter-09) - 自定义扩展
- [第十章：实战案例](/chapter-10) - 真实项目应用

## 快速开始

1. 安装 Superpowers 到你的编码代理
2. 阅读第一章了解核心理念
3. 跟随第二章配置你的环境
4. 开始实践！
